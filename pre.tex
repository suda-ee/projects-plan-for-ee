\section{数字电压表}
设计并制作一块数字电压表。要求掌握双积分~A/D~转换器的工作原理，掌握集成
化双积分~A/D~转换器~ICL7135~和部分常用器件的正确使用方法。\\
基本量程：$\unit{-2}{\volt}\sim\unit{+2}{\volt}$。

\section{二位显示数字计数器}
设计一个计数器电路，要求能实现十进制两位正、倒计数功能，可在~0--99~中预
置初始值，并显示计数值。

\section{可调直流正负电源}
\begin{itemize}
\item 设计并制作一台连续可调的正负直流稳压电源
\item 主要技术指标与要求~(对正或负各部分而言)
    \begin{itemize}
    \item 输出直流电压~$V_{\rm O}=3\sim \unit{12}{\volt}$
    \item 最大输出电流~$I_{\rm LM}=\unit{1}{\ampere}$
    \item 输出纹波电压小于~\unit{5}{\milli\volt}，稳压系数~$S_{\rm
    r}\leq 5\times 10^{-3}$，输出内阻小于~\unit{0.1}{\ohm}
    \end{itemize}
\end{itemize}
